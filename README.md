# Time Stop: Reborn

An unofficial continuation of [MegaTimeStop](https://github.com/Mega32K/MegaTimeStop), ported to Minecraft 1.21.1 on NeoForge.

[中文说明](README-zh_cn.md)

## Building

Requires JDK 21.

```powershell
.\gradlew build
```

The mod JAR is produced under `build/libs/`.
