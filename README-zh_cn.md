# 时停：重生

超绝时停模组 [MegaTimeStop](https://github.com/Mega32K/MegaTimeStop) 的非官方延续版，基于 Minecraft 1.21.1 与 NeoForge 移植。

[English README](README.md)

## 构建

需要 JDK 21。

```powershell
.\gradlew build
```

构建产物位于 `build/libs/` 目录。
